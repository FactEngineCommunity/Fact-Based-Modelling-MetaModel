# Fact-Based-Modelling-MetaModel

A metamodel for sharing Fact-Based Models, such as Object-Role Modeling and FCO-IM.

# Documentation

Read the docs at: [Fact-Based Modelling MetaModel](https://factenginecommunity.github.io/Fact-Based-Modelling-MetaModel/)

# Folder Descriptions

## [XSD]

Stores the latest XSD for the FBM Exchange MetaModel.

## [ClassesMappingToTheXSD]

Stores classes, in different programming languages, that map to the desired XSD.

## [Examples]

Contains example Fact-Based Models exported as XML and conforming to the latest XSD.

## [Test Models - Different Applications]

Models from different Facct-Based Modeling tools that can be used to generate FBM Exchange MetaModel XML documents.

## [Documentation]

Further documentation on Fact-Based Modelling and the FBM Exchange MetaModel.
